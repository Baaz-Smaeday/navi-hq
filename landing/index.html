Run Tests

Run npm test, see pass/fail with colour-coded output. Build your project and catch errors before deploying.

Setup

Ready in 3 minutes

No cloud servers. No API keys. Just your laptop and phone.

1

Clone & setup

Clone the repo and run the setup wizard. It finds your projects, connects to Supabase (free), and builds your dashboard.

git clone github.com/Baaz-Smaeday/navi-hq && bash v2/setup.sh
2

Listener starts automatically

The setup installs a background service that runs on boot. Your laptop is always ready — no manual start needed.

3

Open on your phone

Open the dashboard URL on your phone and add to home screen. Looks and feels like a native app. Done.
