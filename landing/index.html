Up and running in 3 minutes

1

Clone & setup

Clone the repo and run the setup wizard. It finds your projects, connects to Supabase, and generates your dashboard.

git clone https://github.com/Baaz-Smaeday/navi-hq && cd navi-hq && bash v2/setup.sh

2

Start the listener

The setup installs a LaunchAgent that runs automatically on boot. Your laptop is always ready for commands.

3

Open on your phone

Deploy the dashboard to GitHub Pages (free) and open it on your phone. Add to home screen for a native app feel.
